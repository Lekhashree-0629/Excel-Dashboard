# Walmart Dynamic Retail Dashboard
## Excel Interactive Sales & Performance Analysis

This repository contains an advanced Excel Dashboard designed to analyze Walmart’s retail performance across regions, segments, categories, customers, and product lines.
The dashboard is fully interactive using Pivot Tables, Slicers, Charts, and Excel Formulas, allowing users to drill down into key business metrics.

## Dashboard Overview
### Top KPIs

The top panel highlights essential business metrics:

- Total Sales: ₹1,26,42,502

- Total Profit: ₹14,67,457

- Total Quantity Sold: 1,78,312

- Total Orders: 51,290

- Profit Margin: 11.61%

Each KPI contains a small trendline to show performance over time.

## Filters & Slicers

The right section includes powerful filters for deep analysis:

- Segment (Consumer, Corporate, Home Office)

- Market (APAC, EMEA, EU, US, Canada, LATAM)

- Category / Sub-Category

- Region / State

- Ship Mode

- Customer Name

- Product ID

- Order Priority

- Order Year (2019–2024)

These slicers make the dashboard fully dynamic — adjusting every chart instantly.

## Purpose of the Dashboard

This dashboard helps the business:

- Analyze country & region-level performance

- Identify top customers & products

- Track category-level profitability

- Compare historical sales (2019–2024)

- Understand buying patterns across segments

- Improve decision-making with interactive filters
